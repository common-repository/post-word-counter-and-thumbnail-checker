Simple Post Word Counter and Check which post has thumbnail or not.

== Description ==

Post Word Counter and Thumbnail Checker is a smart plugin which shows the total words of a post content and check which post has thumbnail or not. So, you can easily find out the total words of a post and the including or excluding the thumbnail of the post.

Features Include:

* Show Total Number of Words of each Post.
* Show the Featured Image (thumbnail) of each Post.
* No extra settings are required
* Provide extra information about your Posts.
* It is very light-weight

== Installation ==
This section describes how to install the plugin and get it working.

e.g.

* Go to Plugin Menu from your dashboard.
* Click on Add New
* Now upload “post-words-count.zip” file and click install now.
* Or you can search with the plugin name \"Post Word Counter and Thumbnail Checker\" in search box.
* Finally active it and fun with it.
